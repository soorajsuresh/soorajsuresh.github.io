3.1 Absolute Value

3.1.1 Definition The real number $|x|$ is called the absolute value of $x$. If $x\ge0$, then \[|x|:=x.\] If $x<0$, then \[|x|:=-x.\]
Example If $x=9$, then \[|x|=|9|=9.\] If $x=-9$, then \[|x|=|-9|=-(-9)=9.\]

3.2 Distance

Definition 3.2.1 The real number $|x-y|$ is called the distance between $x$ and $y$.
Example If $x=-5$ and $y=8$, then the distance between $x$ and $y$ is \[|x-y|=|-5-8|=|-13|=13.\] If $x$ is any real number, then $|x|$ is the distance between $x$ and $0$, since $|x|=|x-0|$.
