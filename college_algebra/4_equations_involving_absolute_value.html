4 Equations Involving Absolute Value

4.1 The Three Types of Equations Involving Absolute Value

We will solve three types of equations involving absolute value, by applying the definition of absolute value.

When the absolute value is assumed to be equal to a positive number (e.g. $5$), we apply the definition of absolute value to find the two solutions.

Example Find all solutions to $|x|=5$. Since $x$ is a real number, $x\ge0$ or $x<0$. If $x\ge0$, then by definition of absolute value, $|x|=x$, so $x=5$. If $x<0$, then by definition of absolute value $|x|=-x$, so $-x=5$, so $x=-5$. More simply, if $|x|=5$, then, by definition of absolute value, $x=5$ or $x=-5$.

When the absolute value assumed to be equal to $0$, we apply the definition of absolute value to find the single solution.

Example Find all solutions to $|x|=0$. By the definition of absolute value, $x=0$.

When the absolute value is assumed to be equal to a negative number (e.g. $-3$), we apply the definition of absolute value to justify that there are no solutions.

Example Find all solutions to $|x|=-3$. By the definition of absolute value, there is no real number whose absolute value is negative, so the equation $|x|=-3$ has no solutions.

In summary:

  • $|$stuff$|=$positive yields two solutions.
  • $|$stuff$|=0$ yields one solution.
  • $|$stuff$|=$negative yields no solutions.

4.2 Examples of The First Type

In the following six examples, we apply the definition of the absolute value to get two linear equations, each of which we solve as usual.

Example Find all solutions to $|x+3|=7$. By the definition of absolute value, $x+3=7$ or $x+3=-7$. By subtracting $3$ from both sides, $x=4$ or $x=-10$.

Next, we increase the level of sophistication again.

Example Find all solutions to $\left|\frac23x+5\right|=2$ . By the definition of absolute value, $\frac23x+5=2$ or $\frac23x+5=-2$. By subtracting $5$ from both sides, $\frac23x=-3$ or $\frac23x=-7$. By multiplying both sides by $3$ and dividing both sides by $2$, $x=-\frac{9}2$ or $x=-\frac{21}2$.

As we further increase the level of sophistication, we isolate the absolute value before proceeding as we did in the previous examples.

Example Find all solutions to $|5x-1|+8=9$. By subtracting $1$ from both sides, $|5x-1|=1$. By the definition of absolute value, $5x-1=1$ or $5x-1=-1$. By adding $1$ to both sides, $5x=2$ or $5x=0$. By dividing both sides by $5$, $x=\frac25$ or $x=0$.

We increase the level of sophistication again.

Example Find all solutions to $2\left|8x-3\right|+5=10$. By subtracting $5$ from both sides, $2\left|8x-3\right|=5$. By dividing both sides by $2$, $\left|8x-3\right|=\frac52$. By the definition of the absolute value, $8x-3=\frac52$ or $8x-3=-\frac52$. By adding $3$ to both sides, $8x=\frac{11}2$ or $8x=\frac12$. By dividing both sides by $8$, $x=\frac{11}{16}$ or $8x=\frac1{16}$.

Note, in the following example, the right side is negative, but we should isolate the absolute value before concluding that there are no solutions to the equation.

Example Find all solutions to $6-3\left|2x+7\right |=-9$. By subtracting $6$ from both sides, $-3\left|2x+7\right |=-15$. By dividing both sides by $-3$, $\left|2x+7\right |=5$. Note that the right side of the equation became positive. Next, by the definition of the absolute value, $2x+7=5$ or $2x+7=-5$. By subtracting $7$ from both sides, $2x=-2$ or $2x=-12$. By dividing both sides by $2$, $x=-1$ or $x=-6$.

Let us increase the level of sophistication one last time.

Example Find all solutions to $\frac{\frac{25}3\left|\frac{\frac{12}5x-3}2 \right|-78}4=-7$. By multiplying both sides by $4$, ${\frac{25}3\left|\frac{\frac{12}5x-3}2 \right|-78}=-28$. By adding $78$ to both sides, $\frac{25}3\left|\frac{\frac{12}5x-3}2 \right|=50$. By dividing both sides by $25$ and multiplying both sides by $3$, $\left|\frac{\frac{12}5x-3}2\right|=6$. By applying the definition of absolute value, $\frac{\frac{12}5x-3}2=6$ or $\frac{\frac{12}5x-3}2=-6$. By multiplying both sides of each equation by $2$, $\frac{12}5x-3=12$ or $\frac{12}5x-3=-12$. By adding $3$ to both sides of each equation, $\frac{12}5x=15$ or $\frac{12}5x=-9$. By dividing both sides of each equation by $12$ and multiplying by $5$, $x=\frac{25}4$ or $x=-\frac{15}{4}$.

4.3 Examples of The Other Types

We conclude our discussion with a more sophisticated example involving the equations of the other two types that we discussed.

Example Find all solutions to $2\left|3x-8 \right|+\Omega=-4$ when
  1. $\Omega=-4$.
  2. $\Omega=-\frac{15}{16}$.
By assuming $\Omega=-4$, $2\left|3x-8 \right|-4=-4$. By adding $4$ to both sides, $2\left|3x-8 \right|=0$. By dividing both sides by $2$, $\left|3x-8 \right|=0$. By the definition of absolute value $3x-8 =0$. By adding $8$ to both sides, $3x=8$ By dividing both sides by $3$ $x=\frac83$. Next, by assuming $\Omega=-\frac{15}{16}$, $2\left|3x-8 \right|-\frac{15}{16}=-4$. By adding $\frac{15}{16}$ to both sides, $2\left|3x-8 \right|=-\frac{1}{16}$. By dividing both sides by $2$, $\left|3x-8 \right|=-\frac{1}{32}$. By the definition of absolute value, there is no real number whose absolute value is negative, so the equation $\left|3x-8 \right|=-\frac{1}{32}$ has no solutions.
