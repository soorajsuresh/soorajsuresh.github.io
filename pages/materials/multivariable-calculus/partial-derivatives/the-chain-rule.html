The Chain Rule

If $y=f(x)$ is a differentiable function of $x$ and $x=g(t)$ is a differentiable function of $t$, then $y$ is a differentiable function of $t$ and

$\frac{dy}{dt} = \frac{dy}{dx} \frac{dx}{dt}$
.

We call $t$ the independent variable and $y=f(g(t))$ the dependent variable. We can call $x=g(t)$ the intermediate variable.

For $z=f(x,y)$, $x=g(t)$, $y=h(t)$, we call $z$ the dependent variable, $t$ the independent variable, and $x$ and $y$ the intermediate variables.

Chain Rule, One Independent Variable, Two Intermediate Variables: If $z=f(x,y)$ is differentiable and $x=g(t)$, $y=h(t)$ are differentiable, then $z=f(g(t),h(t))$ is differentiable and
$\frac{dz}{dt} = \frac{\partial z}{\partial x}\frac{dx}{dt} + \frac{\partial z}{\partial y}\frac{dy}{dt}$
.
If $z = x^2 - y^2$, $x = \cosh t$, and $y = \sinh t$, then find $\frac{dz}{dt}$.
If $z = x^2 y$, $x = \operatorname{sech}(t-7)$, $y = \sqrt{t+2}$, then find $\left.\frac{dz}{dt}\right|_{(1,3)}$.
If $z=f(x,y)$ is differentiable at $(x_0,y_0)$, and $x=g(t)$, $y=h(t)$ are differentiable at $t_0$, with $x_0=g(t_0)$, $y_0=h(t_0)$, then
$\left.\frac{dz}{dt}\right|_{(x_0,y_0)} = \left.\frac{\partial z}{\partial x}\right|_{(x_0,y_0)} \left.\frac{dx}{dt}\right|_{t_0} + \left.\frac{\partial z}{\partial y}\right|_{(x_0,y_0)} \left.\frac{dy}{dt}\right|_{t_0}$
.
If $z=f(x,y)$, $x=a^2(t)$, $y=2^{b(t)}$, then find $\frac{dz}{dt}$.

For $w=f(x,y,z)$, $x=g(t)$, $y=h(t)$, $z=i(t)$, we call $w$ the dependent variable, $t$ the independent variable, and $x$, $y$, $z$ the intermediate variables.

Chain Rule, One Independent Variable, Three Intermediate Variables: If $w=f(x,y,z)$ is differentiable and $x=g(t)$, $y=h(t)$, $z=i(t)$ are differentiable, then
$\frac{dw}{dt} = \frac{\partial w}{\partial x} \frac{dx}{dt} + \frac{\partial w}{\partial y} \frac{dy}{dt} + \frac{\partial w}{\partial z} \frac{dz}{dt}$
.
Find $\frac{dw}{dt}$ if $w=xz-y^2$, $x=\cos t$, $y=\cosh t$, $z=\sqrt{t}$.

For $w=f(x,y,z)$, $x=g(r,s)$, $y=h(r,s)$, $z=i(r,s)$, we call $w$ the dependent variable, $r$ and $s$ the independent variables, and $x$, $y$, $z$ the intermediate variables.

Chain Rule, Two Independent Variables, Three Intermediate Variables: If $w=f(x,y,z)$, $x=g(r,s)$, $y=h(r,s)$, $z=i(r,s)$ are differentiable, then
$\frac{\partial w}{\partial r} = \frac{\partial w}{\partial x}\frac{\partial x}{\partial r} + \frac{\partial w}{\partial y}\frac{\partial y}{\partial r} + \frac{\partial w}{\partial z}\frac{\partial z}{\partial r}$,
$\frac{\partial w}{\partial s} = \frac{\partial w}{\partial x}\frac{\partial x}{\partial s} + \frac{\partial w}{\partial y}\frac{\partial y}{\partial s} + \frac{\partial w}{\partial z}\frac{\partial z}{\partial s}$.
Chain Rule, Two Independent Variables, Two Intermediate Variables: If $z=f(x,y)$, $x=g(r,s)$, $y=h(r,s)$ are differentiable, then
$\frac{\partial z}{\partial r} = \frac{\partial z}{\partial x}\frac{\partial x}{\partial r} + \frac{\partial z}{\partial y}\frac{\partial y}{\partial r}$,
$\frac{\partial z}{\partial s} = \frac{\partial z}{\partial x}\frac{\partial x}{\partial s} + \frac{\partial z}{\partial y}\frac{\partial y}{\partial s}$.
If $w = \sinh(z)$, $z = f(x,y)$, $x = \ln(rs^2)$, $y = \ln \sqrt{r/s}$, then find $\frac{\partial z}{\partial r}$ and $\frac{\partial z}{\partial s}$.
Find $\frac{dy}{dx}$ if $x^2 + 3xy + y^2 = \arctan(xy) - y^3$.
Formula for Implicit Differentiation: If $f(x,y)$ is differentiable and $f(x,y)=0$ defines $y$ as a differentiable function of $x$, then
$\frac{dy}{dx} = -\frac{f_x}{f_y}$
for any $(x,y)$ such that $f_y(x,y)\ne 0$.
Find $\frac{dy}{dx}$ if $x^2 + 3xy + y^2 = \arctan x - y^3$.
