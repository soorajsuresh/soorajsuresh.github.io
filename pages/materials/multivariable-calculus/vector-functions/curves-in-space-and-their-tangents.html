Curves in Space and Their Tangents

As a particle moves in space over an interval of time $I$, its coordinates can be defined parametrically by
$x=f(t),\ y=g(t),\ z=h(t),\ t\in I.$
The set of all points $(x,y,z)$ for $t\in I$ make up a curve in space that is called the particle's path. The position vector of the particle at time $t$ (the vector that initiates at $O$ and terminates at the position of the particle $P(x,y,z)$ at time $t$) is
$\r(t)=\bracks{f(t)}\i+\bracks{g(t)}\j+\bracks{h(t)}\k,\ t\in I.$
The functions $f$, $g$, and $h$ are called component functions of $\r$.
A vector-valued function (or vector function) on a domain $D$ is a function that assigns a vector to each element in $D$. The graph of a vector function is the curve that it traces out. When discussing vector-valued functions, real-valued functions are called scalar functions.

The equation above defines $\r$ as a vector function of the real variable $t$ on the interval $I$.

Let $\r$ be a vector function defined by $\r(t)=\bracks{f(t)}\i+\bracks{g(t)}\j+\bracks{h(t)}\k$ with domain $D$. Let L be a vector in space and $t_0$ be a number. Saying that the limit of $\r$ as $t$ approaches $t_0$ is L and writing
$\lim_{t\to t_0}\r(t)=\textbf{L}$
means that for every number $\epsilon>0$, there exists a corresponding number $\delta>0$ such that for all $t\in D$,
$\abs{\r(t)-\textbf{L}}<\epsilon\text{ whenever }0<\abs{t-t_0}<\delta.$
Let $\textbf{L}=L_1\i+L_2\j+L_3\k$.
$\lim_{t\to t_0}\r(t)=\textbf{L}$
if and only if
$\lim_{t\to t_0}f(t)=L_1,\ \lim_{t\to t_0}g(t)=L_2,\ \lim_{t\to t_0}h(t)=L_3.$
A vector function $\r$ is called continuous at a number $t=t_0$ in its domain if
$\lim_{t\to t_0}\r(t)=\r(t_0).$
If $\r$ is continuous at every point in its domain, then $\r$ is called continuous.
A vector function is continuous at $t=t_0$ if and only if each of its component functions are continuous at $t=t_0$.

Let $\r(t)=\bracks{f(t)}\i+\bracks{g(t)}\j+\bracks{h(t)}\k$ be the position vector of a particle moving along a curve in space, and suppose $f$, $g$, and $h$ are differentiable functions of $t$. Let $\Delta t$ represent a (positive) change in time, and denote the difference between the particle's position $P$ at time $t$ and $Q$ at time $t+\Delta t$ as

$\Delta\r=\r(t+\Delta t)-\r(t).$

As $\Delta t$ approaches $0$, $Q$ approaches $P$ along the curve, and the secant line $PQ$ approaches a tangent line.

The vector function $\r$ is differentiable at $t$ if $f$, $g$, and $h$ are differentiable at $t$, and the derivative of $\r$ at $t$ is
$\r'(t)=\lim_{\Delta t\to0}\frac{\r(t+\Delta t)-\r(t)}{\Delta t}.$
If the vector function $\r$ is differentiable at $t$, then
$\ddx[\r][t]=\bracks{\ddx[f][t]}\i+\bracks{\ddx[g][t]}\j+\bracks{\ddx[h][t]}\k.$
Find $\ddx[][t]\r(t)$ if
$\r(t)=\parens{\ln{\sqrt[5]{\frac{\parens{2t^2+1}^7}{\parens{t-1}^2\e^{-2t}}}}}\i+\parens{\cosh t}^{\sqrt t}\j+\parens{t\sec(t^3-1)}\k.$

The curve traced by a vector function $\r$ is called smooth if $\ddx[\r][t]$ is continuous and never $\bm0$. A curve formed by connecting finitely many smooth curves is called piecewise smooth.

If $\r$ is the position vector of a particle moving along a smooth curve in space, then $\v=\ddx[\r][t]$ is the particle's velocity vector. If $\v$ is nonzero, then it is tangent to the curve and its direction is called the direction of motion. The magnitude of $\v$ is called the particle's speed, and $\a=\ddx[\v][t]$ is the particle's acceleration vector.
