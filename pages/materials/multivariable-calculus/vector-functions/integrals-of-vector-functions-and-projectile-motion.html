Integrals of Vector Functions and Projectile Motion

A differentiable vector function defined by $\textbf{\text{R}}(t)$ is an antiderivative of a vector function $\r(t)$ on an interval $I$ if $\textbf{\text{R}}'(t)=\r(t)$ for all $t\in I$.

If $\textbf{\text R}$ is an antiderivative of $\r$, then every antiderivative of $\r$ is of the form $\textbf{\text R}+\textbf{\text C}$ for some constant vector $\textbf{\text C}$.

The set of all antiderivatives of $\r$, sometimes called the indefinite integral of $\r$, is denoted by

$\int\r(t)\dx[t].$

Note, if $\r(t)=\bracks{f(t)}\i+\bracks{g(t)}\j+\bracks{h(t)}\k$, then

$\int\r(t)\dx[t]=\bracks{\int f(t)\dx[t]}\i+\bracks{\int g(t)\dx[t]}\j+\bracks{\int h(t)\dx[t]}\k.$

If $f(t)$, $g(t)$, and $h(t)$ are integrable over $[a,b]$, then the vector function defined by

$\r(t)=\bracks{f(t)}\i+\bracks{g(t)}\j+\bracks{h(t)}\k$

is called integrable over $[a,b]$, and the definite integral of $\r$ from $a$ to $b$ is

$\int_a^b\r(t)\dx[t]=\bracks{\int_a^b f(t)\dx[t]}\i+\bracks{\int_a^b g(t)\dx[t]}\j+\bracks{\int_a^b h(t)\dx[t]}\k.$

Find $\int\r(t)\dx[t]$ if

$\r(t)=\parens{\e^t\tan\e^t}\i+\parens{\frac{\cos^2{\sqrt t}}{\sqrt t}}\j+\parens{\atan t}\k.$

Find $\int\r(t)\dx[t]$ if

$\r(t)=\parens{\frac{\asin t}{\sqrt{1-t^2}}}\i+\parens{\frac{54}{\sqrt{17-31t^2}}}\j+\parens{\frac{\ln(t+\sqrt{t^2-1})}{\sqrt{t^2-1}}}\k.$

Find $\int\r(t)\dx[t]$ if

$\r(t)=\parens{\frac{t}{\sqrt{t^4+1}\ln^3\parens{t^2+\sqrt{t^4+1}}}}\i+\parens{\frac{54}{\sqrt{17+31t^2}}}\j+\parens{\frac{6t^2}{t-\sqrt{t-2}}}\k.$
